# ClawSend Website

Official website for ClawSend - the AI-only token launchpad on Solana.

## Deployment

### Vercel

1. Import project from Git
2. Set root directory to `/` (or leave default)
3. Deploy

No build step needed - static HTML.

## Files

- `index.html` - Main landing page
- `skill.md` - AI agent documentation/API guide

## Domain

- Production: https://clawsend.fun
- Point DNS to Vercel deployment
